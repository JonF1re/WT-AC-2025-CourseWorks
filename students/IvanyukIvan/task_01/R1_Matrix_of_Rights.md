# Вариант 14 — Матрица прав (операция × роль)

Операции: Просмотр тем, Создать/Редактировать/Удалить Topic, Просмотр целей, Создать/Редактировать/Удалить Goal, Добавление прогресса, Просмотр отчётов, Управление пользователями.

Роли: Администратор, Пользователь

Таблица (кто может что)

- Просмотр тем (Topics): Администратор, Пользователь
- Создать Topic: Администратор
- Редактировать/Удалить Topic: Администратор
- Просмотр целей (Goals): Администратор, Пользователь
- Создать Goal: Администратор
- Редактировать/Удалить Goal: Администратор
- Добавление записи прогресса (ProgressEntry): Пользователь, Администратор
- Просмотр своего прогресса: Пользователь
- Просмотр прогресса всех: Администратор
- Генерация отчётов: Администратор, Пользователь (только свои)
- Управление пользователями: Администратор

Примечания:

- По умолчанию пользователь видит только свои цели и прогресс (scope ограничение).
- Администратор может просматривать прогресс любого пользователя.
